All About Github

Q. What is a version control system?

A. A class of systems responsible for managing changes to computer programs, documents, large web sites, or other collections of information.


Q. What is Git?

A. A free and open source distributed version control system


Q. What is GitHub?

A. A company offering a cloud-based Git repository that helps developers store, manage, track and control changes to their code.


Q. What is the difference between Git and GitHub?

A. Git is a version control system, while GitHub is what manages Git repositories in the cloud.


Q. Who started GitHub and how was it started?

A. Chirs Wanstrath and PJ Hyett, it was started after the founders made enough improvements to Ruby on Rails to launch GitHub


Q. What company owns it now?

A. Microsoft.


Q. How much does a GitHub account cost?

A. The GitHub Team plan starts at $4 per user/month. GitLab Ultimate will set you back $99 per user/month versus $21 per user/month for GitHub Enterprise.


Q. What is the octocat?

A. The mascot of the source-code hosting service GitHub.
