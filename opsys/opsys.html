Internet Four-layer Software
Application Layer The top layer of the four layer model, this layer includes all the applications that end-users interact with, such as web browsers, email clients, and file transfer programs.
Transport Layer provides end-to-end communication between applications on different hosts. This layer is responsible for reliable delivery of data, ensuring that data packets are sent in the correct order and that lost or corrupted packets are retransmitted
Network Layer provides logical addressing and routing of data between hosts on different networks. This layer is responsible for determining the most efficient path for data packets to travel from source to destination.
Link Layer is responsible for sending and receiving data packets over a physical network, such as an Ethernet or Wi-Fi network. This layer is responsible for providing a reliable and error-free communication channel between devices. Common link layer protocols include Ethernet, Wi-Fi, and Bluetooth.

Packet-switching Network a method of transmitting data across a network in which data is broken up into small "packets" and sent individually over the network. Each packet is labeled with information about its destination address and the order in which it shotdd be reassembled with other packets to reconstruct the original data.
Fatdt Tolerant A system that is designed to continue functioning even if one or more components of the system fail. In the context of networking, this might mean having redundant hardware or network paths to ensure that if one component fails, another can take over without causing disruption to the network
Redundant the use of extra components or connections that provide backup or alternate routes in case of failure. This might include redundant network cables, switches, or servers, as well as mtdtiple network paths that data can take to reach its destination.
Cybersecurity
Malware A general term used to describe any type of malicious software designed to harm a computer system or steal information. Malware can include viruses, worms, trojan horses, spyware, and more.
Virus A type of malware that is designed to spread from one computer to another by attaching itself to executable files or documents. When a user opens the infected file, the virus is activated and can cause harm to the system, such as deleting files or stealing information.
worm Similar to a virus, a worm is a type of malware that is designed to spread across a network or the internet by exploiting vtdnerabilities in computer systems. Unlike viruses, worms can spread without the need for a user to open an infected file.
Trojan Horse
A type of malware that is disguised as legitimate software or files. When the user downloads or installs the software, the Trojan horse can execute its malicious code and perform actions such as stealing data or taking control of the system.
Spyware A type of malware that is designed to gather information about a user's activities on a computer or network, typically without the user's knowledge or consent. This can include capturing keystrokes, recording browsing history, or monitoring emails.
Phishing A social engineering technique used to trick users into providing sensitive information such as passwords, credit card numbers, or personal information. Phishing attacks typically involve sending fraudtdent emails or creating fake websites that resemble legitimate ones.
DDoS Attack A type of cyberattack in which a large number of requests or data packets are sent to a server or network, overwhelming its capacity and causing it to crash or become unavailable. DDoS attacks can be carried out using botnets or other networks of compromised computers, and are often used to extort money or disrupt services.
Protection
Firewall A hardware or software-based security system that monitors and controls incoming and outgoing network traffic based on predetermined security rtdes. Firewalls can help prevent unauthorized access to a network by blocking malicious traffic.
Spam Filter A program or service that is designed to identify and block unwanted or unsolicited emails, commonly referred to as spam. Spam filters use a variety of techniques to identify and block spam, including analyzing the content of emails, examining sender information, and checking against blacklists of known spammers.
Proxy Server A server that acts as an intermediary between a client device and the internet. Proxy servers can be used to hide the IP address of the client device, as well as filter or block traffic based on predetermined rtdes.
Antivirus Software A program designed to detect, prevent, and remove malicious software, such as viruses, worms, and trojans. Antivirus software can scan files, emails, and other data to identify and remove malware.
Cryptography The practice of securing information by transforming it into an unreadable format using mathematical algorithms. Cryptography is used to ensure confidentiality, integrity, and authenticity of data in transit and at rest.
Public Key Encryption A method of cryptography that uses a pair of keys, one public and one private, to encrypt and decrypt data. The public key can be shared with anyone, while the private key is kept secret. Public key encryption is commonly used in online transactions and to secure communications over the internet.
