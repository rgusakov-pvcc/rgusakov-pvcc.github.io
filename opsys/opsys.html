Operating Systems Terminology

TERM DEFINITION
operating system the software that supports a computer's basic functions, such as scheduling tasks, executing applications, and controlling peripherals.
job queue A job queue is a architecture that faciliates task in a ordered fashion which is done syncronously
FIFO A FIFO queue is a queue that operates on a first-in, first-out (FIFO) principle. This means that the request (like a customer in a store or a print job sent to a printer) is processed in the order in which it arrives.
interactive processing A processing method in which each action by a system user causes response from the program or the system. Contrast with batch processing.
time-shared multiprocessing time-sharing, in data processing, method of operation in which multiple users with different programs interact nearly simultaneously with the central processing unit (CPU) of a large-scale digital computer.
