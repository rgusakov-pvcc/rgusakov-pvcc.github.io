job queue A job queue is a architecture that faciliates task in a ordered fashion which is done syncronously
FIFO A FIFO queue is a queue that operates on a first-in, first-out (FIFO) principle. This means that the request (like a customer in a store or a print job sent to a printer) is processed in the order in which it arrives.
interactive processing A processing method in which each action by a system user causes response from the program or the system. Contrast with batch processing.
time-sharing Time-sharing enables many people, located at various terminals, to use a particular computer system at the same time. Multitasking or Time-Sharing Systems is a logical extension of multiprogramming. Processor’s time is shared among multiple users simultaneously is termed as time-sharing.
multiprogramming the running of two or more programs or sequences of instructions simultaneously by a computer with more than one central processor.
application software software designed to handle specific tasks for users. Such software directs the computer to execute commands given by the user and may be said to include any program that processes data for a user.
system software software that is designed to run a computer's hardware and application programs.
utlity software software which includes all systems and programs on a computer system that maintain its functionality
graphical user interface (GUI) a visual way of interacting with a computer using items such as windows, icons, and menus, used by most modern operating systems.
shell A shell is a user interface for access to an operating system's services. Most often the user interacts with the shell using a command-line interface (CLI). The terminal is a program that opens a graphical window and lets you interact with the shell.
kernel kernel is central component of an operating system that manages operations of computer and hardware. It basically manages operations of memory and CPU time. It is core component of an operating system.
device driver a device driver is a special kind of software program that controls a specific hardware device attached to a computer. device drivers are essential for a computer to work properly.
file manager a file manager or file browser is a computer program that provides a user interface to manage files and folders.
memory manager a memory manager is a software utility that operates in conjunction with the operating system.
virtual memory virtual memory is a memory management technique where secondary memory can be used as if it were a part of the main memory.
boot strapping in computing, the term bootstrap means to boot or to load a program into a computer using a much smaller initial program to load in the desired program, which is usually an OS.
ROM read-only memory is a non-volatile memory type. this means it receives data and permanently writes it on a chip, and it lasts even after you turn off your computer. the data is coded to not be overwritten, so it's used for things like your printer software or your startup programs.
deadlock a situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resource, resulting in both programs ceasing to function.
spooling to spool (which stands for "simultaneous peripheral operations online") a computer document or task list (or "job") is to read it in and store it, usually on a hard disk or larger storage medium so that it can be printed or otherwise processed at a more convenient time
super user (or administrator) a user of a computer system with special privileges needed to administer and maintain the system; a system administrator.
multi-factor authentication (MFA) an authentication method that requires the user to provide two or more verification factors to gain access to a resource such as an application, online account, or a VPN. MFA is a core component of a strong identity and access management (IAM).
